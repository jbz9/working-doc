#### 无缓冲 Chan 的发送和接收是否同步?

```
ch := make(chan int)    无缓冲的channel由于没有缓冲发送和接收，需要同步.
ch := make(chan int, 2) 有缓冲channel，不需要发送和接收操作同步，会有阻塞
```
